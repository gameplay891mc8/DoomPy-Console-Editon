# 😈 DOOM: CONSOLE EDITION // G Studio-Team 😈

[EN] **DOOM: Console Edition** is a unique text-based 2D shooter designed specifically to run inside the terminal console. The project went from a turn-based text game to a dynamic real-time action!

[RU] **DOOM: Console Edition** — это уникальный текстовый 2D-шутер, созданный специально для запуска внутри консоли терминала. Проект прошёл путь от пошагового текстового тира до динамичного экшена в реальном времени!

## 📅 Changelog / История версий

### 🚀 REBOOT ERA: Version v0.0.1 (Current / Текущая)
**[EN]** Full project reboot from scratch using a real-time engine!
* **Curses Engine:** Completely dropped endless pauses and the need to press `Enter` after every action. The game reads keys instantly!
* **Real-time Monster AI:** Demons don't wait for your turn anymore. They step forward every 1.2 seconds. If you hesitate, you get eaten.
* **Interactive Menu:** Language selection (English / Русский) directly in the terminal using the arrow keys.
* **Dynamic Feedback:** Added shotgun muzzle flash animation (`💥💥💥` flashes red for a split second) and a system beep sound.

**[RU]** Полный перезапуск проекта со свежей головы с переходом на движок реального времени!
* **Движок Curses:** Полный отказ от бесконечных пауз и необходимости постоянно нажимать `Enter`. Игра считывает клавиши мгновенно на лету!
* **Реальное время (ИИ монстров):** Демоны больше не ждут твоего хода. Они делают шаг вперед каждые 1.2 секунды. Замешкался — тебя съели.
* **Интерактивное меню:** Выбор языка (Русский / English) прямо в терминале с помощью стрелочек на клавиатуре.
* **Динамический фидбек:** Добавлена визуальная вспышка выстрела дробовика (`💥💥💥` окрашивается в красный цвет на долю секунды) и системный звуковой сигнал (Beep).

### ⏳ ARCHIVED ERA: Classic Console Edition (Old / Старая)
**[EN]** The very first historic build of the game where it all started.
* **Turn-based gameplay:** The game moved like a text strategy via standard `input()`.
* **Text commands:** You had to type letters and press Enter for every action (e.g., `f` for fire).

**[RU]** Первая историческая сборка игры, с которой все начиналось.
* **Пошаговый геймплей:** Игра двигалась по принципу текстовой стратегии через стандартный ввод `input()`.
* **Текстовые команды:** Для каждого действия нужно было прописывать буквы в консоль (например, `f` для огня).

## 🕹️ Controls / Управление (Reboot-0.0.1):
* `Arrow Keys Up/Down` / `Стрелочки Вверх/Вниз` — Navigation / Навигация в меню.
* `Enter` — Confirm language / Подтверждение выбора.
* `F` / `f` — Shoot / Огонь из дробовика.
* `R` / `r` — Reload / Перезарядка (+5 ammo).
* `Q` / `q` — Quit game / Выход.

### 📱 Android (PyDroid 3):
**[EN]** If you just press the yellow "Play" button inside PyDroid 3, the game will crash. Run it strictly via the Terminal in the side menu: type `python main.py` and hit Enter on your keyboard.
**[RU]** Если вы просто нажмете желтую кнопку «Play» внутри PyDroid 3, программа вылетит. Запускайте строго через Terminal в боковом меню: введите `python main.py` и нажмите Enter.

### 🐧 Linux (Linux Master Build):

```bash
python3 main.py
```

```cmd
pip install windows-curses
```
